## Vertical Cards

* **Type:** Table
* **Section:** available
* **Actions:** cell long pressure
* **Image required:** yes

## How to integrate

* To use a list form template, the first thing you'll need to do is create a YourDatabase.4dbase/Resources/Mobile/form/list folder.
* Then drop the list form folder into it.
